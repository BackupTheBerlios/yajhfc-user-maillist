Thanks for the bug report. It seems that the launcher script (execyajhfc.vbs) 
from version 0.3.9 has a bug which sometimes causes it to set an incorrect 
class path for the FOP plugin.

So, please replace c:\program files\yajhfc\execyajhfc.vbs with the updated 
version from ftp://ftp.berlios.de/pub/yajhfc/temp/execyajhfc.vbs and try 
again.

If it still does not work, it would be helpful if you could mail be a sample 
ODT file with which you are having problems.
