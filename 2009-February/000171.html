Am Freitag, 13. Februar 2009 schrieb Sorin Toma:
> Hello,
> I'm trying to add gv and tiff2pdf into the installer package, in order
> to make my life easier when I will start deploying Yajhfc  in our
> company,  but it seems like I'm missing something.
> I used innounp (http://innounp.sourceforge.net/) to unpack the
> installer, but, even without any modification, if I'm trying to repack
> it with innosetup (http://www.jrsoftware.org/isinfo.php), I'm unable to
> find some functions like Check: "RedmonNotInstalled", Check:
> "NoYajHFCPrinter" and BeforeInstall: "InstallYajHFCPort()"
> Can somebody give me a hint about what am I doing wrong?

To install the fax printer the YajHFC setup uses some custom code written in 
InnoSetup's RemObject Pascal language ([code] section in the .iss file).
I have never used innounp, but it seems that it does not extract this code 
correctly, so that these functions are missing.

Personally (especially for internal deployment), I would rather create some 
kind of "wrapper" around the setup executables of YajHFC, gs and tiff2pdf.
If you want to do that, you can use "/SILENT" switch to do a silent 
installation with the YajHFC (and tiff2pdf) setup for example (see the Inno 
Setup help or 
http://unattended.sourceforge.net/InnoSetup_Switches_ExitCodes.html for 
details).


Hope that helps,
Jonas
