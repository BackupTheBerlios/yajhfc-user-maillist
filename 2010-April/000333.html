Am Montag, 12. April 2010 schrieb Stephen Kerr:
> Is it possible for MS Excel to automatically send parameters to the Yajhfc
> client i.e. Fax Number?
>
> I have Excel templates that have different fax numbers, and it would
> enable me to fax them without having to input the fax number each time.

Please do not post such questions to the yajhfc-announce list, but use 
yajhfc-user instead.

Now to your question:
Currently this is not possible (it is very difficult to reliably extract 
information from PostScript documents produced by a printer driver).

It should be possible to write an Excel macro passing that information to 
YajHFC, however, if you find a way to convert the Excel sheet to 
PDF/PostScript using the macro.
Then you would start YajHFC and pass both the document and the fax number on 
the command line.

Hope that helps
Jonas
