This is a bug in Java, see:
http://bugs.sun.com/bugdatabase/view_bug.do?bug_id=6449933

This means I can not really fix it.
According to the bug report at bugs.sun.com, a possible workaround is using 
another look&feel.
To change the look&feel without starting YajHFC, edit the settings file and 
replace the line "lookAndFeel=\!system\!" 
with "lookAndFeel=\!crossplatform\!".
